Canon PIXMA MG6250 - macOS printer and scanner install kit

Unofficial scripts to install printing and scanning on current macOS using Canon’s last official macOS 10.13 (High Sierra) drivers. Not affiliated with or endorsed by Canon.

  • Repository - source, issues, and full README
  • README - quick start and caveats
  • Canon Europe - MG6250 drivers - official downloads
  • Blog note - context on modern macOS

Clone the repo and follow the scanner and printer sections in the README. Network printing uses Bonjour/IPP; the documented scanner path uses USB and Image Capture.
